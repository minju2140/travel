# 🚀 앱 실행 가이드

## 빠른 시작

### 1단계: 백엔드 실행

새 터미널을 열고 다음 명령을 실행하세요:

```powershell
cd E:\yeong\couple-trip-expense\backend
npm run dev
```

백엔드 서버가 `http://localhost:5000`에서 실행됩니다.

### 2단계: 프론트엔드 실행

**새로운 터미널을 하나 더 열고** 다음 명령을 실행하세요:

```powershell
cd E:\yeong\couple-trip-expense\frontend
npm run dev
```

프론트엔드가 `http://localhost:5173`에서 실행됩니다.

### 3단계: 브라우저에서 확인

브라우저를 열고 `http://localhost:5173`로 접속하세요!

## 🎯 첫 사용 가이드

### 1. 사용자 1 (첫 번째 사람)
1. 회원가입 페이지에서 계정 생성
2. 로그인
3. "초대 코드 생성하기" 버튼 클릭
4. 생성된 6자리 코드를 파트너에게 공유 (예: `ABC123`)

### 2. 사용자 2 (파트너)
1. **다른 브라우저** 또는 **시크릿 모드**에서 `http://localhost:5173` 접속
2. 회원가입 페이지에서 계정 생성
3. 로그인
4. "초대 코드로 참여하기" 버튼 클릭
5. 파트너가 공유한 코드 입력

### 3. 여행 시작하기
1. 첫 번째 사용자 화면에서 "새 여행" 버튼 클릭
2. 여행 정보 입력:
   - 여행 이름: 예) "도쿄 여행"
   - 시작일/종료일 선택
   - 방문 국가: 예) "일본"
3. 생성 버튼 클릭

### 4. 지출 기록하기
1. 생성된 여행 카드 클릭
2. 우측 하단의 **+** 버튼 클릭
3. 지출 정보 입력:
   - 지출 항목: 예) "라멘 식사"
   - 금액: 예) 1200
   - 통화: JPY 선택
   - 카테고리: 식비 선택
   - 지불 유형: "내가 냄" 또는 "공동 지출" 선택
4. 저장하기 클릭

👉 **실시간 동기화 확인**: 파트너 화면에서도 즉시 지출이 나타나는지 확인하세요!

### 5. 통계 확인하기
- **대시보드 탭**: 예쁜 차트로 지출 현황 확인
- **지출 내역 탭**: 날짜별 상세 지출 목록
- **정산 탭**: 누가 누구에게 얼마를 보내야 하는지 자동 계산!

## 💡 팁

- **두 명의 사용자 테스트**: 같은 컴퓨터에서 테스트하려면 한 쪽은 일반 브라우저, 다른 한 쪽은 시크릿 모드를 사용하세요
- **실시간 동기화**: 한 사람이 지출을 추가하면 다른 사람 화면에 즉시 반영됩니다
- **다양한 통화**: 여러 나라를 여행할 때 각 나라의 통화로 기록하면 자동으로 원화로 환산됩니다

## 🛠️ 문제 해결

### 백엔드가 시작되지 않는 경우
```powershell
cd E:\yeong\couple-trip-expense\backend
# 데이터베이스 다시 생성
npx prisma migrate reset --force
npx prisma generate
npm run dev
```

### 프론트엔드 오류가 있는 경우
```powershell
cd E:\yeong\couple-trip-expense\frontend
# node_modules 재설치
rm -r node_modules
npm install
npm run dev
```
